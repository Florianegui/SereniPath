# Poster SereniPath — Session Poster

## Utilisation

1. **Ouvrir** `poster-serennipath.html` dans un navigateur (Chrome, Firefox, Edge)
2. **Remplacer** la zone "Insérer une capture d'écran" :
   - Faire une capture de l'app (carte, itinéraires, lieux sécurisants)
   - Ou modifier le HTML : remplacer le `div.demo-placeholder` par `<img src="ta-capture.png" style="width:100%; border-radius:12px;">`
3. **Exporter** :
   - **Impression** : Ctrl+P → Imprimer ou "Enregistrer au format PDF"
   - **Format** : A1 (59×84 cm) ou A0 (84×119 cm) selon la session

## Personnalisation

- Modifier le texte directement dans le fichier HTML
- Ajouter ton nom / équipe dans le footer
- Changer les couleurs dans le `<style>` (variables `--seren-*`)
